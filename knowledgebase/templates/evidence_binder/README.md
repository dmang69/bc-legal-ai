# Evidence binder index

Suggested tabs:

1. Chronology  
2. Lease / tenancy agreement  
3. Notices and service proof  
4. Photos / videos (with hashes)  
5. Correspondence  
6. RTB decision / audio  
7. Financials  
8. Privilege log (not produced)  

Each exhibit maps to an EvidenceNode `node_id` + `doc_hash`.
