# Doctrine Cards (Living)

## Res judicata / issue estoppel (working)

| Field | Content |
|-------|---------|
| Elements | Prior final decision; same parties (or privies); same issue/cause (framework-dependent) |
| When applies | Same dispute re-litigated |
| When fails | New issues; not final; different parties |
| Authority | Verify leading SCC/BCCA cases before citing |
| BC / RTA note | Prior RTB decision may ground *res judicata* argument — fact-specific |

## Natural justice / procedural fairness (working)

| Field | Content |
|-------|---------|
| Elements | Notice; opportunity to be heard; unbiased decision-maker (context varies) |
| When applies | Administrative and some court-adjacent decisions |
| Authority | Common law; Baker factors for fairness content |
| RTB note | No notice / no chance to respond → review/JR pathway analysis |

## Functus officio (working)

| Field | Content |
|-------|---------|
| Plain | Decision-maker finished — generally can’t reopen |
| Exceptions | Statute may allow review/reconsideration |
| RTA note | Confirm review provisions on BC Laws for the proceeding |
