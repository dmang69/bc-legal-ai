# Hugging Face Documentation

Operational docs for Hugging Face setup, publishing, and governance.

Docs in this folder should explain how to create and publish HF assets without storing secrets or real client data.

Planned docs:

- `SETUP.md` — account, token, and repo setup.
- `PUBLISHING.md` — publish workflows for spaces/datasets/models.
- `GOVERNANCE.md` — data and legal safety rules.
