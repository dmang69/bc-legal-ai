# Hugging Face Datasets

Dataset publishing workspace.

## Intended public dataset

```text
dmang69/bc-legal-ai
```

## Allowed contents

- Skills.
- Lexicon/glossary materials.
- Synthetic examples.
- Official-source metadata and verification logs where redistribution is permitted.
- Templates that do not contain real client data.

## Prohibited contents

- Real client files.
- Live litigation records.
- Privileged communications.
- Unredacted transcripts, affidavits, exhibits, or strategy notes.
- Bulk legal corpora unless licence/terms review authorizes publication.

## Dataset governance

Every dataset release should include:

- Dataset card.
- Source inventory.
- Licence/terms notes.
- Confidential-data scan result.
- Synthetic-data declaration.
