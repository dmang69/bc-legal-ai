# Cross-Citation Matrix — Blank Template

Track which works in your portfolio cite which. Populate for existing
works, then check for missing citations.

```
                │ Work A │ Work B │ Work C │ Work D │ Work E │
────────────────┼────────┼────────┼────────┼────────┼────────┤
Work A cites    │   ─    │        │        │        │        │
Work B cites    │        │   ─    │        │        │        │
Work C cites    │        │        │   ─    │        │        │
Work D cites    │        │        │        │   ─    │        │
Work E cites    │        │        │        │        │   ─    │
```

## Filling In the Matrix

Mark each cell:
- `✓` — cites, and citation is load-bearing
- `~` — cites, but only in passing
- `?` — should cite but doesn't
- blank — no citation and none needed

## Diagnostics

Once filled:

**Isolated works** — rows/columns with few marks. Either genuinely
independent or under-integrated with the portfolio.

**Hub works** — rows/columns with many marks. Load-bearing works
that the rest of the portfolio builds on.

**Missing citations (`?` cells)** — the action list. Each `?` is a
citation to add on next revision.

**Asymmetric citations** — A cites B but B doesn't cite A. Sometimes
correct (B predates A). Sometimes a gap (they should mutually cite).

## Maintenance

Update after every new work is finished. The matrix should never be
more than a work behind current state.
