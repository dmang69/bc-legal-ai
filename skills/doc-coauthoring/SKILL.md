## One-line mission

Build docs that work for **other readers** (and other Claudes), not only the authors.

## Stages

1. **Context Gathering** — meta-questions, dump, 5–10 clarifying Qs  
2. **Refinement & Structure** — section scaffold; Qs → brainstorm → curate → draft → edit  
3. **Reader Testing** — fresh Claude Q&A; fix blind spots
