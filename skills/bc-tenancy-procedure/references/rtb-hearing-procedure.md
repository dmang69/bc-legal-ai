# RTB Hearing Procedure (Working Notes)

> Verify all form codes, fees, and deadlines on the official RTB site before filing.

## Channels

Online portal · Mail · Service BC — confirm current options.

## Application hygiene

- Exact names from agreement  
- Full address + unit designation  
- Numbered issues  
- Dollar claims quantified  
- RTA section per claim  
- Evidence by deadline (often ~14 days pre-hearing — **`[VERIFY]`**)  
- Witnesses listed  

## Hearing modes

| Mode | Notes |
|------|--------|
| Participatory | Phone/video/in-person — most contested matters |
| Written | Smaller claims / consent |
| Direct Request | Uncontested / certain claims if no response — **`[VERIFY eligibility]`** |

## During hearing

1. Opening (30–60 seconds)  
2. Evidence by tab  
3. Cross with documents  
4. Closing IRAC + precise order  

## After decision

- Read effective date and what is ordered  
- Monetary order → enforcement path  
- Defects → internal review / JR (see advocacy Module 1)
