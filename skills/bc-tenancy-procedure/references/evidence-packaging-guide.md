# Evidence Packaging Guide (RTB)

> Legal information only. Confirm current RTB evidence rules and deadlines.

## Goal

Make the arbitrator’s job easy: **claim → exhibit → date → conclusion**.

## Hierarchy reminder

Primary docs → contemporaneous messages → dated media → third-party → live witnesses → hearsay.

## 10-tab kit (copy into every file)

1. Agreement  
2. Condition inspections  
3. Payments  
4. Correspondence chronology  
5. Photos/videos  
6. Repair requests/responses  
7. Notices + service proof  
8. Third-party records  
9. Verified CanLII decisions  
10. RTA + Policy Guidelines excerpts  

## Labelling

- Exhibits **A, B, C…**  
- Photos: Exhibit · date · what · where  
- Every chronology line ends with `(Ex. _)`

## Timeline format

```
YYYY-MM-DD — Event — (Ex. A)
YYYY-MM-DD — Event — (Ex. B p.2)
```

## Quality checks before upload

- [ ] No undated photos without explanation  
- [ ] Screenshots show full header (number/date) where possible  
- [ ] Bank records match claimed amounts  
- [ ] Service proof for every notice relied on  
- [ ] Redact SIN / account numbers as appropriate
