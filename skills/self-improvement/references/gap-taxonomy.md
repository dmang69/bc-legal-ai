# Gap Taxonomy

| Type | Example (tenancy suite) | Fix |
|------|-------------------------|-----|
| Factual | Wrong RTA section for deposits | REPLACE in substantive Module 7 |
| Procedural | Missed evidence deadline step | ADD to procedure Module 3 |
| Depth | Mentioned renoviction without first-refusal remedy | Expand Module 2 |
| Scope | No MHPTA coverage | NEW MODULE or advanced skill |
| Trigger | JR question didn’t load judicial-review skill | DESCRIPTION update |
| Format | Submissions without IRAC | Update procedure Module 5 |
| Integration | Evidence pack without RTA cites | Hard handoff procedure → substantive |

## Detection signals → types

| Signal | Likely type |
|--------|-------------|
| User: “actually the section is…” | Factual |
| User: “you forgot the deadline” | Procedural + Critical |
| User re-asks with more detail | Depth / Anticipation |
| Wrong skill activated | Trigger |
| User rewrites entire draft structure | Format |
