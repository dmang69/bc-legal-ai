# Court / Tribunal Decision Reading Protocol

## First pass (fast)

1. Court/tribunal, date, parties, disposition (allowed/dismissed)  
2. Procedural history (one paragraph)  
3. Issues listed by the decision-maker  
4. Outcome per issue  

## Second pass (slow)

```
STANDARD OF REVIEW: 
ISSUE:              
TEST:               
FINDINGS:           
HOLDING:            
REASONING (RATIO):  
OBITER:             
```

## RTB-specific

- Identify notice type, forms, RTA sections cited  
- Verify statute pins on **BC Laws** before relying (`bc-legislation-admin`)  
- Note findings of credibility, good faith, service  
- Flag ignored evidence (Module 6 — unaddressed / fact-law gap)  

## Output for JR file

Must-address points + weaknesses to exploit + admissions + ratio usable on review.
