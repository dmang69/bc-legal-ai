# Affidavit Reading Protocol

## Sort every paragraph

| Label | Meaning |
|-------|---------|
| **PK** | Personal knowledge |
| **I&B** | Information and belief (source required ideally) |
| **ARG** | Argumentative / legal conclusion (often improper) |
| **DOC** | Exhibits attached |

## Checks

- [ ] Double hearsay without source  
- [ ] Argument disguised as fact  
- [ ] Exhibits actually support the paragraph  
- [ ] Internal contradictions across paragraphs  
- [ ] Admissions useful to the other side  

## Response use

Extract admissions → Module 9 “ADMISSIONS TO PRESERVE”.  
Attack ARG and unsupported I&B as weight, not necessarily “strike” unless rules allow.
