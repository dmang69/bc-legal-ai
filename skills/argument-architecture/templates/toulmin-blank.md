# Toulmin Blank Template

```
CLAIM
─────
[What you're arguing — one precise sentence.]


GROUNDS
───────
[Evidence supporting the claim. List each independent piece.]
  - [Grounds 1]
  - [Grounds 2]
  - [Grounds 3]


WARRANT
───────
[The reasoning link. WHY do the grounds support the claim?

Warrant type: [Causal / Analogical / Definitional / Statistical / Authority]

Statement: [Explicit articulation of the reasoning move.]]


BACKING
───────
[Support for the warrant itself. Where does the warrant come from? Is it
established in the literature, by prior work, by logical necessity?]


QUALIFIER
─────────
[Under what conditions does the claim hold? What are the boundaries of
its scope?]

  Holds when: [conditions]
  Does not extend to: [out-of-scope cases]


REBUTTAL
────────
[Where would the claim fail? What conditions would defeat it?

  The claim would fail if [conditions]
  Those conditions [do / do not] obtain here because [reasons]]
```
