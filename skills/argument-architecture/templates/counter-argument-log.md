# Counter-Argument Log — Blank Template

For each load-bearing claim, log the counter-argument protocol.

```
CLAIM: [what you're arguing]

COUNTER 1
─────────
Objection:      [the strongest opposing position]
Who would make it: [judge / opposing counsel / peer reviewer / skeptical reader]
Basis:          [what makes this objection strong]
Response:       [your best response]
Concession?:    [if partial, what and why it doesn't defeat the claim]


COUNTER 2
─────────
Objection:      [second-strongest]
Basis:          [same]
Response:       [same]
Concession?:    [same]


COUNTER 3
─────────
Objection:      [third-strongest]
Basis:          [same]
Response:       [same]
Concession?:    [same]


UNADDRESSED
───────────
Objections you thought of but chose not to address in the document:
  - [objection] — [why you're not addressing it: too weak, out of scope,
    would open a bigger issue, etc.]

Note: any unaddressed objection is a risk. Track them so if they come up,
you have a considered response ready.
```
