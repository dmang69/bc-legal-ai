# Argument QA Checklist — Final Sweep

Run before any argument leaves your hands.

```
STRUCTURE
─────────
□ Main claim in intro matches main claim in conclusion
□ Each section serves the main claim
□ Bridges between sections name the logical relationship
□ No section is unmoored from the argument spine

CLAIMS
──────
□ Every claim is either evident, cited, or reasoned to
□ Every citation actually says what it's cited for
□ Confidence markers match actual confidence level

REASONING
─────────
□ Every deductive chain has valid form (modus ponens / tollens / hyp syll)
□ Every inductive chain acknowledges its probabilistic nature
□ Every abductive chain acknowledges alternative explanations
□ No affirming the consequent
□ No denying the antecedent
□ No undistributed middle

COUNTER-ARGUMENTS
─────────────────
□ Strongest objection is addressed
□ Objection is steel-manned before rebuttal
□ Concessions are bounded (don't defeat the main claim)

FALLACIES
─────────
□ No ad hominem
□ No straw man
□ No slippery slope
□ No illegitimate appeal to authority
□ No appeal to popularity
□ No circular reasoning
□ No false dilemma
□ No hasty generalization
□ No post hoc
□ No equivocation
□ No red herring
□ No tu quoque

PROSE
─────
□ Every paragraph passes the wince test
□ No padding
□ No conclusory language ("clearly", "obviously") where argument is needed
□ Bridge sentences name the relationship

HOSTILE READER TEST
───────────────────
□ Simulated a skeptical reader looking for reasons to disagree
□ Fixed any weak spots they'd find
□ Read the argument aloud (in your head) — does it flow?
```

Any "no" → revise before submitting.
