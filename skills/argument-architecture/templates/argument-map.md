# Argument Map — Blank Template

```
                        MAIN CLAIM
                             │
       ┌─────────────────────┼─────────────────────┐
       │                     │                     │
   SUB-CLAIM 1          SUB-CLAIM 2           SUB-CLAIM 3
   [text]               [text]                [text]
       │                     │                     │
   ┌───┴───┐             ┌───┴───┐             ┌───┴───┐
 EVID 1.1 EVID 1.2     EVID 2.1 EVID 2.2     EVID 3.1 EVID 3.2
 [text]   [text]       [text]   [text]       [text]   [text]

Cross-support:
  EVID [X] also supports SUB-CLAIM [Y]

Counter-branch:
                     COUNTER-CLAIM
                     [strongest objection]
                          │
                     RESPONSE
                     [why the counter fails]
```

## Map Discipline Checks

Before drafting from the map:

```
□ Does each sub-claim independently support the main claim?
□ Would removing any sub-claim weaken the main claim materially?
□ Does each piece of evidence actually support the sub-claim it's under?
□ Are cross-supports marked?
□ Is the strongest counter-claim mapped?
□ Is the response to the counter-claim mapped?
□ Are any sub-claims restatements of each other? (Merge)
□ Are any sub-claims restatements of the main claim? (Cut)
```

Once the map passes these checks, draft from it. The draft adds
transitions, tone, and specifics — but preserves the skeleton.
