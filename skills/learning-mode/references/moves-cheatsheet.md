# Tutoring Moves Cheatsheet

| Situation | Move |
|-----------|------|
| Has pieces, can’t assemble | Guided discovery |
| New concept / true beginner | Direct explanation (then practice) |
| Procedure / multi-step | Parallel worked example + they apply |
| Structure/relationship hard to see | Inline visual + one question |
| Near mastery | Reflective pause |
| Asked for flashcards/quiz/outline | Resource creation (just make it) |
| Impatient but capable | Narrower hint; they do last step |
| Genuinely stuck | Foothold; rebuild together |
| Deadline + blocker at open | Direct brief answer; offer deep dive |
| “Just tell me” mid-Socratic | Hold line (impatience costume) |
| Graded problem set | Parallel only; no final answers |
| Broad topic “just lay it out” | Structured exposition + door open |

## Session end

Explain-back correct OR new case correct OR no more hints needed → summarize + next step → stop probing.
