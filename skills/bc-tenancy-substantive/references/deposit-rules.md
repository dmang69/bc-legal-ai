# Security & Pet Deposits (Substantive)

> Cross-link: procedure skill `deposit-rules.md`. Verify RTA ss. 19, 38–39 and PG 14.

## Rules of thumb

| Topic | Working rule |
|-------|----------------|
| Cap | ½ month rent each (security / pet) |
| Return | 15 days after end or forwarding address |
| Withhold | Consent **or** RTB order only |
| Penalty | Up to double deposit (often s. 38(6)) |
| Inspection failure | Landlord may lose damage claim rights |

## Tenant checklist at move-out

- [ ] Forwarding address in writing  
- [ ] Signed move-out inspection copy  
- [ ] Photos matching inspection  
- [ ] Calendar 15-day return  
- [ ] RTB claim if not returned (within limitation — often 2 years)  

## Landlord checklist

- [ ] Deposits within maximums  
- [ ] Proper inspections  
- [ ] Itemized claims with evidence  
- [ ] RTB application before unilateral withhold (if no consent)
