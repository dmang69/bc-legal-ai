# Retaliation Doctrine (RTA s. 47.1)

> Legal information only. Verify current s. 47.1 text.

## Core idea (plain)

A landlord cannot punish a tenant for standing up for legal rights (repairs, RTB filing, reporting, organizing, etc.).

## Proof pattern

```
[Date] Tenant exercises right (repair demand / RTB / complaint)
[Date] Landlord issues notice / raises rent / harasses / enters illegally
```

Timing alone is not automatic proof — but close timing is powerful **circumstantial** evidence.

## Combine with

- Illegal charge refusal → eviction  
- Repair requests → renoviction notice  
- RTB application → cause notice  

## Remedies

Set aside notice · damages/compensation · other orders as available · possible human rights dual-file if protected ground.

## Labels

`[FACT]` complaint date · `[FACT]` notice date · `[ARGUMENT]` s. 47.1 · `[REMEDY]` void notice
