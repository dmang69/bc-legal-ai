# Legal knowledge (authoritative layer)

Store retrieved materials with **currency metadata**, not only free-text blobs.

| Source | Use |
|--------|-----|
| BC Laws | Statutes and regulations **only** (never CanLII for BC statute text) |
| RTB / gov.bc.ca | Forms, Rules of Procedure, policy guidelines |
| BC Courts | Practice directions, forms, judgments, filing |
| CanLII | Decisions, citation relationships |
| SCC / federal | When engaged |

Each authority record should capture fields in `architecture.schemas.AuthorityRecord`.

Local official extracts currently live under `../legislation/` (RTA, verification log).
